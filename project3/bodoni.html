Bodoni Speciman Poster








The Bodoni type speciman poster was the first typography project I worked on at OTIS. The challeges of this poster was working with the scale of the poster (24in x 36in) and working with glyphs.





The Process

This project required a lot of attention to craft and detail. I started with creating ligatures and working soley with tracing paper and pencils. It wasn't until the final process of this project were I got to work digitally and create the Bodoni type speciman poster.
